Servientrega Mercancía Premier

== Description ==

Con este plugin podrás configurar tu cuenta de Servientrega para que se calculen los costos de envío.

= Supported Countries =
 * Colombia


== Installation ==

= Automatic installation =

1. Vaya a **Escritorio » Plugins » Añadir nuevo**
2. En el formulario de búsqueda digite **"Servientrega Mercancía Premier"**
3. Cuando la búsqueda devuelva el resultado, haga clic en el botón **Instalar Ahora** 
4. Finalmente, haga clic en el botón **Activar** 

= Manual Installation = 
1. Descargue el plugin desde esta página haciendo clic en el botón **Download**
2. Vaya a **Escritorio » Plugins » Añadir nuevo**
3. Selecciones el botón **Subir Plugin** 
4. Haga clic en **Seleccionar archivo** y seleccione el archivo que quiere descargar
5. Haga clic en **Instalar ahora** y luego active el plugin

= Consideraciones especiales =

Para que este plugin funcione correctamente es necesario tener instalado el pluging States, Cities, and Places for WooCommerce e instalar el plugin 	
Administración de Pedidos Servientrega para la gestión de los envíos.
